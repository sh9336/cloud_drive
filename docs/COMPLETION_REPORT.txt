🔐 SECURITY FEATURES
═══════════════════════════════════════════════════════════════════════════════

Threat Model 1: Path Traversal
  Attack: "upload_to": "../admin/"
  Defense: Exact node name matching only
  Status: ✅ BLOCKED

Threat Model 2: Arbitrary S3 Paths
  Attack: "upload_to": "tenants/other-id/files/"
  Defense: Only template-defined nodes allowed
  Status: ✅ BLOCKED

Threat Model 3: Cross-Tenant Access
  Attack: Access other tenant's files
  Defense: JWT tenant_id enforced, all paths isolated
  Status: ✅ IMPOSSIBLE

Threat Model 4: Bypassing Permissions
  Attack: Upload to disabled folder
  Defense: can_upload flag enforced
  Status: ✅ BLOCKED

Threat Model 5: Uploading Directories
  Attack: "upload_to": "folder_that_expects_files"
  Defense: allow_files flag enforced
  Status: ✅ BLOCKED
